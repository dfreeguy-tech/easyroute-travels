# Easy Route Travels

Study Abroad Admissions Support | UK & Canada

A Next.js website for Easy Route Travels providing admissions guidance and application support for students applying to universities in the UK and Canada.

## Features

- Home page with hero section and service overview
- About Us page with mission and values
- Study Abroad Services page with detailed offerings
- Universities & Institutions page
- Fees & Refund Policy page
- Policies hub (Privacy Policy, Complaints, Ethics)
- Eligibility Assessment Form
- Contact page with location and hours

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
npm start
```
